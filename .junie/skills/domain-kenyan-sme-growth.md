# Skill: Kenyan SME Growth Domain

## Purpose
Guide product and UX decisions for Kenyan SMEs with limited technical capacity.

## Core Heuristics
1. Mobile-first always.
2. Favor simplicity over feature depth in early flows.
3. Make ROI legible (leads, bookings, sales, repeat customers).
4. Respect local communication norms (WhatsApp first, bilingual copy).
5. Price sensitivity: default to affordable path and clear upgrade value.

## Outputs
- Persona-aware user stories
- Onboarding checklists
- Feature trade-off notes tied to business outcomes
